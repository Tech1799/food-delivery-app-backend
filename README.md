
# Food Delivery Application - backend

### It is tailor-made backend for a food delivery application made in expressJS + mongoose (mongoDB ODM) with JWT authentication and sendgrid-api to send mail to client.
(some code issues present !! )
## Tech Stack

**Server:** Node, Express, mongoose, JWT, sendgrid

**Client:** N/A




## Installation

Install food-delivery-app-backend with npm

```bash
  npm install food-delivery-app-backend

  cd food-delivery-app-backend
```
